# **App Name**: Treo

## Core Features:

- Role-Based Authentication: Secure login with email, password, and role-based access control.
- Team Management: Core team can create, edit, and manage teams, including adding members with assigned roles and permissions.
- Task Management: Core team assigns tasks to teams/members, tracks progress with status updates and deadlines.
- Document and File Management: Upload, store, and organize files with role-based access permissions, linking files to specific teams and users.
- Team Chat: Enable team-based communication through real-time messaging, with core team members able to access all team chats.
- AI Task Suggester: An AI assistant will act as a tool and will provide suggested tasks, improving the efficiency of planning events
- Dashboard UI: Modern, minimal UI with intuitive navigation, responsive layout for both mobile and desktop, and customizable themes.

## Style Guidelines:

- Primary color: Light blue (#ADD8E6) for a clean, trustworthy look.
- Background color: Very light gray (#F5F5F5) to create a spacious, uncluttered feel.
- Accent color: Soft orange (#FFB347) to highlight important CTAs.
- Font: 'Inter' (sans-serif) for both body and headlines, offering a modern, neutral appearance. Note: currently only Google Fonts are supported.
- Use clear, consistent icons to enhance usability and provide visual cues throughout the app.
- Emphasize a modular layout with clear sections and consistent spacing to aid usability.
- Incorporate subtle animations for feedback and transitions to enrich the user experience.
